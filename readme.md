### ⚡️ 기본템 커맨드 ⚡️️
```yml

/기본템 : 기본템을 지급 받습니다.

/기본템 설정 : 기본템을 설정할 수 있는 GUI를 오픈합니다.

/기본템 초기화 [플레이어] : [플레이어]의 기본템을 초기화합니다.

```
<hr>

### 📑 기본템 펄미션 📑️
